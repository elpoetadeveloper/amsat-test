MOVE-1 First-Move

Catalog number: 39439
Launch Date: November 21, 2013
Status: Operational.

Packet Downlink:  145.970 MHz 1200bps BPSK AX.25
Beacon: 145.970 CW every 60 seconds
Callsign: MOVE1

Dear radio amateurs,

Our satellite transmits signals using the amateur radio frequency of 145.970 MHz and due to its polar orbit can be received by radio amateurs around the world. We would appreciate it if some of you listened for a transmission from our 'little guy' to help us keep track of the satellite's status. Some further information:

  • The on-board transceiver transmits at 145.970 MHz.
  • Due to the nature of the transmit antenna, the signal has no polarisation
  • The satellite alternates between beacon modes every 60 seconds. One beacon is a CW beacon of the satellites designation in morse code ('MOVE1'), the second mode is an AX.25 package containing a few telemetry items.
  • The AX.25 beacon is modulated using Raised Cosine-BPSK.
  • The downlink data rate is 1200 baud.
  • We have a small software tool that can decode the AX.25 beacon and show you the telemetry values contained in the signal. The part of the software that decodes the audio signal was provided by our transceiver vendor ISIS.
  • To receive the beacons set the receiver or transceiver to USB (Upper Side Band) mode.
  • The software expects a BPSK signal with a center frequency of 1600 Hz. Therefore, when using an Upper Side Band (USB) receiver, you must tune 1600 Hz below the actual downlink frequency.
  • The software can only compensate an offset of +/- 700 Hz, so the doppler shift has to be compensated with other means, manually or automatic.

The program MOVE Telemetry Receiver can be downloaded here

To use the software Java Runtime Version 6 or higher needs to be installed. To decode the AX.25 packets, connect the speaker output of your transceiver to the microphone or line-in jack of your computer. If you have any questions or comments regarding the software, just write us an e-mail to the following address: move-amateur-radio@lrt.mw.tum.de

As soon as the software has successfully decoded a packet, the raw values will be written to a text files. The text file will be placed into the users main directory (i.e. Windows: C:/Users// ) and the file name has the following format: move_YYYY-MM-DD.txt. If you receive and decode signals from First-MOVE we would greatly appreciate it, if you could send us the text file or an audio recording by e-mail.

For those interested you can download our telemetry items list here which also includes the conversion formulas for the raw values.
